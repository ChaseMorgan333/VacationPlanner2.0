Installation Instructions

Follow the steps below to install the application on an Android device or emulator.

Option 1: Android Emulator (Recommended)

  1. Open Android Studio.
  2. Launch an Android Emulator using Device Manager.
  3. Drag and drop the downloaded APK file into the running emulator window.
  4. Wait for the installation confirmation message.
  5. Open the app from the emulator’s app drawer.

Option 2: Physical Android Device

  1. Download the APK file to your Android device.
  2. If prompted, enable Install unknown apps for your browser or file manager.
  3. Open the APK file and confirm installation.
  4. Launch the app from your device’s app list.

Note: This application is not distributed through the Google Play Store, so Android may display a security warning during installation.
